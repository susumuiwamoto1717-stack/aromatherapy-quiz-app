## 2. 電力（電験三種）

### 2.1 送配電システムの基礎

#### 2.1.1 直流配電の特徴
DCマイクログリッドでは、従来の交流（AC）配電ではなく直流（DC）配電を採用します。

**直流配電の利点：**
- **変換損失の削減**：AC/DC変換が不要なため、変換損失を最小化
- **高効率**：太陽光発電や蓄電池は本質的に直流であるため、変換ステップを削減
- **電圧降下の抑制**：同じ電力伝送において、DCはACよりも電圧降下が小さい
- **AIデータセンターとの親和性**：サーバーや電子機器は内部でDCを使用するため、直接接続が可能

**直流配電の課題：**
- **遮断器の技術**：DCはゼロクロスがないため、アークの消弧が困難
- **絶縁設計**：DC電圧による絶縁劣化の特性が異なる
- **保護協調**：AC系統とは異なる保護方式が必要

#### 2.1.2 電圧レベルと規格
DCマイクログリッドで一般的に使用される電圧レベル：

- **低圧DC（48V、380V）**：データセンター内の配電
- **中圧DC（750V、1500V）**：マイクログリッド内の主配電
- **高圧DC（±375V、±750V）**：長距離送電や大容量システム

**電験三種で重要な電圧区分：**
- 低圧：直流750V以下、交流600V以下
- 高圧：直流750V超、交流600V超～7000V以下
- 特別高圧：7000V超

### 2.2 電力品質と電圧制御

#### 2.2.1 電圧変動の要因
DCマイクログリッドにおける電圧変動の主な要因：

1. **負荷変動**
   - AIデータセンターの負荷は変動が激しい（CPU使用率に応じて）
   - 突入電流による瞬間的な電圧降下

2. **発電量の変動**
   - 太陽光発電の出力変動（雲の通過など）
   - 蓄電池の充放電状態による電圧変動

3. **系統インピーダンス**
   - 配線抵抗による電圧降下：ΔV = I × R
   - 配線インダクタンスの影響（特に高周波成分）

#### 2.2.2 電圧制御方法
**PCS（パワーコンディショニングシステム）による制御：**

```
電圧制御の基本式：
V_out = V_ref ± ΔV_control

ΔV_control = f(負荷電流, 系統状態, 蓄電池SOC)
```

**制御方式：**
- **定電圧制御（CV制御）**：系統電圧を一定に保つ
- **定電流制御（CC制御）**：充放電電流を制御
- **定電力制御（CP制御）**：出力・入力を一定に保つ

**電験三種で学ぶ電圧変動率：**
```
電圧変動率 = (無負荷電圧 - 定格電圧) / 定格電圧 × 100 [%]
```

### 2.3 保護継電器と保護協調

#### 2.3.1 DC系統の保護課題
DC系統では、AC系統とは異なる保護方式が必要です。

**主な保護要素：**
1. **過電流保護**
   - 短絡電流の検出と遮断
   - DCはACと異なり、短絡電流が指数関数的に増加

2. **過電圧・不足電圧保護**
   - 系統電圧の異常検出
   - 蓄電池の過充電防止

3. **逆電力保護**
   - 蓄電池から系統への逆流防止（必要に応じて）
   - 系統連系時の保護

#### 2.3.2 保護協調の設計
**時限協調：**
```
保護装置の動作時間を段階的に設定
例：負荷側 0.1秒 → 分岐 0.3秒 → 主回路 0.5秒
```

**電流協調：**
```
保護装置の動作電流値を段階的に設定
例：負荷側 200A → 分岐 500A → 主回路 1000A
```

### 2.4 接地方式

#### 2.4.1 DC系統の接地方式
**非接地方式：**
- 系統と大地を絶縁
- 1線地絡時も系統を継続運転可能
- 地絡検出が困難

**1点接地方式：**
- 系統の中性点を1点で接地
- 地絡電流の経路を明確化
- 地絡保護が容易

**2点接地方式：**
- 複数点で接地
- 地絡電流が大きくなるため、保護動作が確実
- 地絡時の影響範囲が広い

**電験三種で重要な接地抵抗：**
- 低圧：100Ω以下（漏電遮断器使用時は500Ω以下）
- 高圧：10Ω以下（特別高圧は1Ω以下）

## 3. 機械（電験三種）

### 3.1 パワーコンディショナー（PCS）の動作原理

#### 3.1.1 PCSの基本構成
PCSは、直流と交流を相互変換する装置です。DCマイクログリッドでは、主に以下の機能を担います：

**主要機能：**
1. **DC/DC変換**：電圧レベルの変換（例：蓄電池48V → 系統380V）
2. **DC/AC変換**：系統連系時の変換
3. **AC/DC変換**：交流系統からの充電
4. **電力制御**：充放電の制御、系統電圧の安定化

#### 3.1.2 変換回路の基礎
**降圧チョッパ（Buckコンバーター）：**
```
出力電圧：V_out = D × V_in
（D：デューティ比、0 < D < 1）
```

**昇圧チョッパ（Boostコンバーター）：**
```
出力電圧：V_out = V_in / (1 - D)
（D：デューティ比、0 < D < 1）
```

**昇降圧チョッパ（Buck-Boostコンバーター）：**
```
出力電圧：V_out = -D × V_in / (1 - D)
（極性が反転する）
```

#### 3.1.3 PWM制御
**パルス幅変調（PWM）の原理：**
- スイッチング周波数：通常10kHz～100kHz
- デューティ比を変えることで出力電圧を制御
- 高周波化により、リアクトル・コンデンサを小型化可能

**電験三種で学ぶスイッチング損失：**
```
P_sw = (V × I × t_sw × f_sw) / 2
（t_sw：スイッチング時間、f_sw：スイッチング周波数）
```

### 3.2 蓄電池システム

#### 3.2.1 リチウムイオン電池の特性
**基本特性：**
- **公称電圧**：3.2V（LiFePO4）、3.7V（NCM）など
- **容量**：Ah（アンペア時）で表記
- **エネルギー密度**：Wh/kg、Wh/Lで表記

**充放電特性：**
```
充電：外部電源から電流を流し込む（電圧上昇）
放電：負荷に電流を供給（電圧降下）

SOC（State of Charge）：充電状態
SOC = (現在の容量 / 最大容量) × 100 [%]
```

#### 3.2.2 バッテリーマネジメントシステム（BMS）
**BMSの主要機能：**
1. **セルバランス制御**
   - 各セル間の電圧差を均一化
   - 過充電・過放電の防止

2. **保護機能**
   - 過電圧保護（OVP）
   - 過電流保護（OCP）
   - 過熱保護（OTP）
   - 過放電保護（UVP）

3. **状態監視**
   - SOC（充電状態）の推定
   - SOH（健康状態）の監視
   - 温度監視

#### 3.2.3 蓄電池の接続方式
**直列接続：**
```
総電圧 = セル電圧 × セル数
例：3.2V × 120セル = 384V
```

**並列接続：**
```
総容量 = セル容量 × 並列数
例：100Ah × 4並列 = 400Ah
```

**直並列接続（バンク構成）：**
```
総電圧 = セル電圧 × 直列数
総容量 = セル容量 × 並列数
総エネルギー = 総電圧 × 総容量
```

### 3.3 インバーター・コンバーター

#### 3.3.1 インバーターの基本動作
**単相インバーター：**
- Hブリッジ回路を使用
- 4つのスイッチで正負の電圧を生成
- PWM制御で正弦波を生成

**三相インバーター：**
- 6つのスイッチ（IGBT、MOSFETなど）を使用
- 120度位相差の三相交流を生成
- ベクトル制御により高精度な制御が可能

#### 3.3.2 コンバーターの効率
**変換効率：**
```
η = (出力電力 / 入力電力) × 100 [%]
  = (V_out × I_out) / (V_in × I_in) × 100 [%]
```

**損失の種類：**
1. **スイッチング損失**：スイッチのオン/オフ時の損失
2. **導通損失**：スイッチのオン抵抗による損失
3. **リアクトル損失**：コイルの銅損・鉄損
4. **コンデンサ損失**：ESR（等価直列抵抗）による損失

**電験三種で学ぶ効率の計算：**
```
総合効率 = η1 × η2 × η3 × ...
（直列接続された機器の総合効率）
```

### 3.4 モーターと発電機の基礎（参考）

#### 3.4.1 直流モーター
DCマイクログリッドでは直接関係しませんが、理解しておくと良い知識：

**基本式：**
```
トルク：T = K × φ × I_a
（K：定数、φ：磁束、I_a：電機子電流）

回転速度：N = (V - I_a × R_a) / (K × φ)
（V：端子電圧、R_a：電機子抵抗）
```

#### 3.4.2 同期発電機
系統連系時に関連：

**基本式：**
```
誘導起電力：E = 4.44 × k × f × N × φ
（k：巻線係数、f：周波数、N：巻数、φ：磁束）

同期速度：N_s = 120 × f / P
（f：周波数、P：極数）
```

### 3.5 DCマイクログリッドにおける実践的知識

#### 3.5.1 PCSと蓄電池の接続
**典型的な構成：**
```
太陽光発電 → DC/DCコンバーター → 系統DCバス
                                      ↓
蓄電池 ← DC/DCコンバーター（双方向） ← 系統DCバス
                                      ↓
AIデータセンター ← DC/DCコンバーター ← 系統DCバス
```

**双方向DC/DCコンバーター：**
- 充電時：系統 → 蓄電池（降圧動作）
- 放電時：蓄電池 → 系統（昇圧動作）
- 制御により自動的に切り替え

#### 3.5.2 AIデータセンターの電力要件
**負荷特性：**
- **定常負荷**：サーバーのベース消費電力
- **変動負荷**：CPU使用率に応じた消費電力の変動
- **突入電流**：サーバー起動時の大きな電流

**電力密度：**
- ラックあたり：5kW～20kW
- データセンター全体：数MW～数十MW

**電圧要件：**
- サーバー内部：12V、5V、3.3Vなど
- ラック配電：48V DC（一般的）
- 系統電圧：380V DC（提案）

#### 3.5.3 システム設計のポイント
**容量設計：**
```
蓄電池容量 = (負荷電力 × 必要時間) / (放電深度 × 効率)
例：(100kW × 4時間) / (0.8 × 0.9) = 556kWh
```

**PCS容量設計：**
```
PCS容量 = max(最大充電電力, 最大放電電力)
例：max(200kW, 150kW) = 200kW
```

**電圧設計：**
```
系統電圧 = 負荷電圧 × (1 + 電圧降下率)
例：380V × 1.05 = 399V（5%のマージン）
```
