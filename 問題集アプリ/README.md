# DCマイクログリッド学習問題集アプリ

電験三種の知識を、わかりやすく学べる四択問題集アプリです。

## 📚 特徴

- **本質を理解できる問題**：単なる暗記ではなく、原理を理解できる問題を厳選
- **小学生でもわかる解説**：専門用語を使わず、身近な例で説明
- **カテゴリー別学習**：基礎知識、電力、機械、蓄電池など、テーマ別に学習可能
- **進捗管理**：正解率とカテゴリー別の結果を表示

## 🚀 使い方

1. `index.html` をブラウザで開く
2. モードを選択：
   - **全問チャレンジ（20問）**：すべての問題に挑戦
   - **10問チャレンジ**：ランダムに10問
   - **5問チャレンジ**：ランダムに5問
   - **カテゴリー別**：特定のテーマだけを学習
3. 四択問題に答える
4. 解説を読んで理解を深める
5. 結果を確認して復習

## 🎯 問題カテゴリー

- **基礎知識**：直流と交流の違い、DCマイクログリッドの基本
- **電力の基礎**：電圧降下、電圧変動率、電圧制御
- **機械の基礎**：PCS、コンバーター、PWM制御
- **蓄電池**：リチウムイオン電池、BMS、接続方式
- **効率**：変換効率、総合効率の計算
- **保護**：過電流保護、保護協調
- **接地**：接地方式の種類と特徴
- **実践**：システム設計、容量計算

## 💡 学習のコツ

1. **間違えた問題を復習**：結果画面でカテゴリー別の正解率を確認
2. **解説をじっくり読む**：小学生でもわかる説明で、本質を理解
3. **繰り返し学習**：同じ問題を何度も解いて、定着させる
4. **学習ガイドと併用**：`DCマイクログリッド学習ガイド_電力と機械.md` と一緒に読むと効果的

## 🔧 カスタマイズ

### 問題を追加する

`questions.js` の `questions` 配列に、以下の形式で問題を追加できます：

```javascript
{
  id: 21,
  category: "カテゴリー名",
  question: "問題文",
  options: [
    "選択肢1",
    "選択肢2",
    "選択肢3",
    "選択肢4"
  ],
  correct: 0,  // 正解のインデックス（0-3）
  explanation: "詳しい解説",
  simpleExplanation: "小学生でもわかる説明"
}
```
